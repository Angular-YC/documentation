# Angular-YC Documentation

Static documentation site for Angular-YC (GitHub Pages).

## Local Development

```bash
npm run serve
```

Serves `public/` at `http://localhost:3001`.

## Structure

- `public/index.html` - landing page
- `public/docs.html` - docs hub
- `public/getting-started/` - install and quick start
- `public/api/` - command and manifest reference
- `public/guides/` - migration guides
- `public/examples/` - example walkthroughs
- `public/reference/` - troubleshooting
