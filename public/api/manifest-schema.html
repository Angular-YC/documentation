Manifest Schema

deploy.manifest.json is produced by angular-yc build and consumed by Terraform.

Top-Level Keys

  • schemaVersion, buildId, timestamp
  • angularVersion, projectName
  • capabilities, routing, artifacts
  • responseCache, environment, deployment

Response Cache Model

Angular-YC uses response caching instead of ISR: HTML/body in Object Storage and metadata/tag TTL in YDB.
