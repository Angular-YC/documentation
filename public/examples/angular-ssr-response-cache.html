Angular SSR + response cache

Reference pattern for caching rendered SSR responses with storage-backed invalidation metadata.

This example demonstrates how rendered response bodies are stored in Object Storage while YDB tracks cache metadata, TTL, and tag-based invalidation pointers.

Use it as a baseline for high-traffic pages where deterministic cache controls matter.
