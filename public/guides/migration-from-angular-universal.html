Migration Guide

Move from Angular Universal to Angular-YC

  1. Keep your existing Angular SSR app structure and Express entry point.
  2. Replace custom deployment scripts with angular-yc analyze/build/upload.
  3. Adopt manifest-driven Terraform module inputs for repeatable environments.
  4. Configure response cache settings where you previously relied on ad-hoc SSR caching logic.
